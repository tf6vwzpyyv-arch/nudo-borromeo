# NeuroPsicoLocos Topological Lab

Laboratorio interactivo de topología, psicoanálisis y visualización 3D.

## Módulos iniciales

- 🌀 Nudo Borromeo
- ∞ Banda de Möbius
- 🍩 Toro
- ♾ Botella de Klein
- 🟡 Sinthome

## Objetivo

Crear una herramienta didáctica para explorar conceptos topológicos utilizados por Lacan, integrando modelos 3D interactivos, explicaciones clínicas y recursos para docencia.
